# coffee CLI

A CLI for coffee.

## Customizing your CLI

Check out the documentation at https://github.com/infinitered/gluegun/tree/master/docs.


## Installation

```shell
$ yarn install
$ npm link
```

## Publishing to NPM

To package your CLI up for NPM, do this:

```shell
$ npm login
$ npm whoami
$ npm lint
$ npm test
(if typescript, run `npm run build` here)
$ npm publish
```
